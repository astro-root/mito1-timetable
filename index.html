iPhone / iPad ホーム画面に時間割を表示

Scriptable アプリを使って、最新の時間割をウィジェットとして常時表示できます。アプリは無料です。

1

Scriptable をインストール

iPhone / iPad の App Store で「Scriptable」を検索してインストールしてください。無料アプリです。

App Store で開く
2

スクリプトを追加する

Scriptable を開き、右上の + ボタン → 右上の ⋯ → Add Script from URL をタップ。
下記の URL を貼り付けて「Add Script」を押してください。

https://gist.githubusercontent.com/astro-root/d6da0ed8655f86409309b4abb92f50b0/raw/timetable.js
3

スクリプトを実行して確認

追加されたスクリプトをタップして開き、▶ ボタンで実行します。
時間割の画像がプレビュー表示されれば成功です。

4

ホーム画面にウィジェットを追加

① ホーム画面を長押し → 左上の + をタップ
② 「Scriptable」を検索して選択
③ サイズは 中（Medium） を選んで「ウィジェットを追加」
④ 追加されたウィジェットをタップ → Script の欄でインポートしたスクリプトを選択
⑤ When Interacting を Run Script に設定するとタップで手動更新できます

💡 ウィジェットの更新頻度は iOS が自動で制御します。時間割が変更されたタイミングではウィジェットをタップして手動更新するか、しばらく待つと最新の内容に切り替わります。
